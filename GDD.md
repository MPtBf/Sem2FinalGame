
Минимальный GDD для первой версии, прототипа:

### 1. Дрон (движение, стрельба).

движение:
- движение - плавный, управляемый полёт. Камера всегда следует за дроном
- управляемость движения клавиатурой, направления взгляда, стрельбы, взаимодействия - мышкой
  
размер: немного менее 2 тайлов

стрельба:
- пули - конечны
- крафтятся при взаимодействии с буром из обработанной руды

комбат:
- на убийство монстра уходит несколько пуль
- монстр может кинуться на игрока и ударить, оттолкнуть, если он приблизится. Потратится здорове игрока
- при сметри игрок возраждается в центре бура (сборка нового дрона)

взаимодействие:
- выкопать руду в стене перед дроном
- закинуть руду в бур на изготовление заплатки
- наложить заплатку на повреждённое место у бура

### 2. Бур (HP, убывание HP от врагов).

размер:
- бур значительно больше игрока. 
- игрок взаимодействует с крафтом в буре через стенку (не в салоне)

HP:
- бур может прожить множество нападений врагов (пусть 30)
- игрок чинит бур заплатками (восстанавливает HP)
- игрок видит HP бура на UI элементе
- при сметри бур останавливается и игроку показывается экран проигрыша

движение:
- бур плавно двигается вперёд, оставляя за собой пещеру (разрушает тайлы перед собой)
- при движении вперёд, с боков вскрываются пещеры в блоках, откуда лезут враги (регулярно, пока пещера не закрыта)
- игрок может остановить и возобновить движение по кнопке

### 3. 1 тип врага (простейший поиск пути к буру — это твой "Средний алгоритм").

размер: немного менее 1 тайла

перемещение
- появляется в открытых пещерах
- паук - выбегает из пещеры, бежит к буру по прямой, сразу цепляется, или бьёт игрока (если подошёл близко)
- при получении урона от игрока отбрасывается назад и бежит аттаковать его
- при пропаже игрока из поля зрения переключается снова на бур

комбат:
- враг требует 3 попаданий от ирока, чтобы погибнуть
- при сметри пропадает
- когда цепляется на бур, повреждает его раз в 2 секунды, пока не сбит игроком

### 4. 1 тип ресурса (просто для зачета DRY/KISS).

ресурс: медь

свойства:
- добывается дроном из открытых буром тайлов, и из пещер
- крафтится в заплатки и пули
- при добыче из тайла руды, разрушение этого тайла, получение ресурса в инвентарь

### 5. Сетка (Grid) вместо "абсолютной разрушаемости" (для KISS).

мир состоит из сетки квадратных тайлов.
для оптимизации генерируются только открытые тайлы "стенок" (вокруг тоннеля после бура и в пещерах)

пещеры:
- пещеры с врагами генерируются случайно при разрушении тайлов буром
- пещеры имеют случайную глубину (пару десятков тайлов) и случайную ширину (пару тайлов тайла)
- при генерации пещеры, в ней появляется несколько врагов
