# Sem2FinalGame («Deep Relic»)

Это Итоговая игра для экзамена по программированию за 2 семестр.

## описание игры (в пару слов):

«Deep Relic — это атмосферный сурвайвал о брошенном после катастрофы в шахте дроне, который должен чинить, улучшать и защищать свой постепенно разрушающийся бур, превращая его в латаную крепость, чтобы пробиться сквозь враждебные недра к цивилизации, которая предпочла о нём забыть»

## Структура кода игры:

### Core - Ядро и управление

- **`main.py`**: Точка входа. Создаёт объект `Game` и запускает его. Больше ничего не делает.
- **`game.py`**: Главный менеджер. Управляет временем (DT), переключает состояния (меню/игра) и связывает `World`, `Renderer` и `Camera` воедино.

### Models

- **`world.py`**: Контейнер. Хранит списки всех врагов, пуль, игрока и бур. Обсчитывает их столкновения (физику) и удаляет мёртвых.
- **`game_object.py`**: Базовый чертёж. Хранит общие поля (HP, позиция, скорость) и базовый метод движения `move()`.
- **`drone.py`**: Логика дрона. Управляет инвентарём (медь) и стрельбой.
- **`drill.py`**: Логика бура. Считает прочность, движется вглубь, "съедает" тайлы сетки перед собой.
- **`enemy.py`**: Искусственный интеллект. Решает, куда бежать (к буру или игроку) и когда атаковать.
- **`map.py`**: Карта мира. Хранит массив тайлов (камень, медь, пустота). Знает, что находится в конкретной точке координат.

### Views

- **`renderer.py`**: Художник. Получает объекты из `World`, спрашивает у `Camera` их экранные координаты и рисует картинки (спрайты) на экран.
- **`camera.py`**: Математик. Вычисляет "сдвиг" мира. Следит за игроком, чтобы он всегда был в центре экрана, и выдаёт остальным модулям правильные координаты для отрисовки.

### Utils - Инструменты

- **`misc.py`**: Вспомогательные функции (например, ключ словаря по значению).
- **`astar.py`**: сложный алгоритм (A\*). Принимает сетку `Grid`, точку А и точку Б — возвращает путь для врага.

## Проблемы, трудности и решения:

*проблемы в разработке:*
- проблема: на карте мира невозможно генерить все тайлы камня в стенах - решение: оставлять "unexplored area", спавнить тайлы камня только вокруг воздуха, расширяя территорию при разрушении камня
- проблема: коллизии баганые - решение: переход на разделение axis

*остальные проблемы:*
- проблема: времени на игру мало - Решение: просто спать 1-1.5 ч ночью
- проблема: кодить не умею - Решение: ТОЛЬКО НЕ ВАЙБКО-... терпеть, учиться, уделять этому освободившиеся 5-7 ч в день (ночь).
- проблема: времени на игру всё равно мало - Решение: simply не готовиться к экзамену по алгему
- проблема: сплю над кодом - Решение: просто спать 7-8 ч ночью
- проблема: время кончилось, экзамен завтра - Решение: включить голову и расставить приоритеты. Сдать на сотку(?)
